# Showdown.js Demo

让你的站点也支持Markdown —— showdown.js

### Showdown.html

模拟CSDN等博客的Markdown编辑器，支持实时预览功能。

页面分为两列，左边为markdown语法输入区域，右边为markdown语法预览区域。

showdown.js版本：1.8.2

### CommentBox.html

查看showdown.js是如何将markdown语法转化为html。

react版本：0.14.5   showdown.js版本：0.3.1
